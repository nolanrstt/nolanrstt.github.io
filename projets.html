InfoTools CRM

C# WPF .NET 8 MySQL MSTest LiveCharts iTextSharp
📝 Présentation du projet

InfoTools est une application de bureau (Client Lourd) développée en C# (WPF) sous le framework .NET 8. Il s'agit d'un logiciel de gestion de la relation client (CRM) conçu pour une PME. Le projet s'inscrit dans une démarche de maintenance évolutive, avec un fort accent sur l'optimisation des données et la cybersécurité.

✨ Fonctionnalités Principales
  • Tableau de Bord Interactif : Visualisation des statistiques de l'entreprise en temps réel via des graphiques (LiveCharts).
  • Gestion Commerciale (CRUD) : Administration unifiée des clients, des prospects et du catalogue de produits IT.
  • Agenda Intelligent : Système de prise de rendez-vous avec prévention automatique des chevauchements d'horaires.
  • Sécurité & Traçabilité : Enregistrement complet des actions utilisateurs (historique et audit) avec sérialisation des données au format JSON.
  • Reporting : Génération automatique de bilans et de statistiques au format PDF.

FG Destruction

HTML5 / CSS3 Tailwind CSS PHP JavaScript API Gouvernement FormSubmit
🎯 Objectif du projet

Ce projet web a pour but de fournir une interface claire et rassurante pour les clients confrontés à des nuisibles dangereux. Il permet d'identifier l'insecte, de consulter une grille tarifaire transparente et de demander une intervention rapide via un formulaire de contact.

📂 Structure & Technologies
  • Design & UI : Utilisation de Tailwind CSS pour un design moderne, responsive et l'intégration d'un Dark Mode sur mesure.
  • Dynamisme PHP : Gestion des variables globales (téléphone, nom de l'entreprise) pour faciliter les futures mises à jour sur toutes les pages.
  • UX & JavaScript : Autocomplétion de l'adresse postale dans le formulaire via l'API Adresse du gouvernement français.
  • Contact : Gestion de l'envoi des e-mails du formulaire sans backend complexe grâce à l'outil FormSubmit.

JobPilot

PHP Laravel MySQL API REST MVC Logs & Audit
💼 Contexte & Objectifs

JobPilot est une plateforme web de gestion d'offres d'emploi développée dans le cadre de ma formation en BTS SIO option SLAM.

L'objectif du projet était de créer une application complète permettant aux entreprises de publier des offres d'emploi et aux candidats de postuler directement en ligne. Le projet inclut également un système d'administration permettant de gérer les utilisateurs et leurs rôles.

⚙️ Fonctionnalités développées
  • Gestion des offres : Création, modification et suppression des annonces (CRUD complet).
  • Authentification & Rôles : Système sécurisé avec Laravel gérant 3 niveaux d'accès : Admin, Entreprise et Candidat.
  • Espace Candidature : Possibilité pour les candidats de postuler directement aux offres.
  • Administration globale : Interface d'administration dédiée pour gérer les utilisateurs, les candidats et les entreprises.
  • API REST : Création d'une API sécurisée pour accéder aux données de la plateforme depuis l'extérieur.
  • Sécurité & Traçabilité : Mise en place de logs d'audit pour suivre les actions importantes des utilisateurs.
